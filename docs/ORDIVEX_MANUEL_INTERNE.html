PARTIE I PRSENTATION & ARCHITECTURE

Chapitre 1 Prsentation Gnrale

1.1 Qu'est-ce qu'OrdiveX ?

OrdiveX est un ERP (Enterprise Resource Planning) complet conu spcifiquement pour la gestion des pharmacies en Afrique. Dvelopp par TrillionX, il fonctionne en mode offline-first (100% hors-ligne) avec synchronisation cloud optionnelle.

1.2 Philosophie de Conception

Principe Description
**Offline-First** L'app fonctionne a 100% sans internet. Les donnes sont stockes localement dans IndexedDB
**PWA** Progressive Web App installable sur PC, tablette et smartphone
**FEFO** First Expired, First Out gestion automatique des lots par date de premption
**Scurit Patient** 30+ interactions mdicamenteuses vrifies a chaque vente
**Multi-devises** GNF, XOF, MAD, EUR, USD configurable par pharmacie
**Mobile Money** Orange Money & MTN MoMo natifs

1.4 Stack Technologique

  • Frontend : HTML5, CSS3, JavaScript ES6+ (Vanilla aucun framework)
  • Base de donnes : IndexedDB (via API native)
  • Cloud : Supabase (PostgreSQL + Auth + Realtime)
  • Icnes : Lucide Icons
  • Hbergement : GitHub Pages
  • Impression : API window.print() avec templates A4 et thermique 80mm

2.4 LiveSync Rafrachissement Automatique

Quand une donne change dans IndexedDB (ex: nouvelle vente), le systme _notifyUIChange() dans db.js rafrachit automatiquement la page active si elle dpend du store modifi :

Page Active Stores couts
`dashboard` `sales`, `saleItems`, `products`, `stock`
`pos` *(jamais rafrachi automatiquement trop risqu)*
`products` `products`
`stock` `stock`, `products`
`sales` `sales`
`caisse` `cashRegister`, `sales`
`settings` `users` *(pas settings pour viter les reloads en boucle)*

Chapitre 3 Installation & Dploiement

3.1 Prrequis

  • Navigateur moderne : Chrome 90+, Edge 90+, Firefox 90+, Safari 15+
  • Espace disque : minimum 500 Mo disponible pour IndexedDB
  • Rsolution cran : minimum 360px (mobile) / recommand 1280px+ (desktop)

3.2 Installation sur PC (PWA)

Ouvrir https://johnnyjonnes78-tech.github.io/Xpharma/ dans Chrome

Cliquer sur l'icne "Installer" dans la barre d'adresse

Confirmer l'installation

L'app apparat comme une application native sur le bureau

3.3 Installation sur Mobile

Ouvrir l'URL dans Chrome Android / Safari iOS

Menu "Ajouter a l'cran d'accueil"

L'app s'installe avec son icne

# Depuis le rpertoire du projet
