# src/

Dossier source du projet. Structure recommandée :

```
src/
├── components/       # Composants réutilisables
├── pages/            # Pages / routes
├── lib/              # Fonctions utilitaires, helpers
├── hooks/            # Custom hooks React
├── types/            # Types TypeScript
├── styles/           # CSS global
└── constants/        # Constantes de l'app
```
