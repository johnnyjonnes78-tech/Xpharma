1. 🏥 Présentation d'OrdiveX

OrdiveX est un ERP (Enterprise Resource Planning) pharmaceutique conçu spécifiquement pour les pharmacies d'Afrique francophone. Il permet de gérer l'intégralité de votre officine depuis un seul outil :

  • 🛒 Ventes — POS ultra-rapide avec scan code-barres
  • 📦 Stocks — Gestion automatique FEFO (First Expired, First Out)
  • 💊 Catalogue — 50 000+ produits avec DCI, formes, et notices
  • 📋 Ordonnances — Enregistrement numérique avec interactions médicamenteuses
  • 👤 Patients — Dossiers complets avec allergies et historique
  • 🚚 Fournisseurs — Commandes, réceptions, évaluation
  • 📊 Analyses — Tableau de bord, métriques financières, tendances
  • 🤖 Naomie — Assistante IA intégrée qui vous guide en temps réel
  • ☁️ Cloud — Synchronisation entre PC, tablette et mobile
  • 📴 Offline-first — Fonctionne à 100% sans internet
💡 Philosophie Offline-First OrdiveX fonctionne SANS internet. Toutes vos données sont stockées localement sur votre appareil. La connexion internet n'est utilisée QUE pour la synchronisation cloud entre plusieurs appareils.

6. 🛒 Point de Vente (POS)

Le POS est le cœur d'OrdiveX. C'est ici que vous réalisez les ventes quotidiennes.

Faire une vente — Procédure pas-à-pas

  1. Rechercher un produit : Tapez le nom, la DCI ou le code-barres dans la barre de recherche. Les résultats apparaissent instantanément.
  2. Ajouter au panier : Cliquez sur la carte du produit. Il s'ajoute au panier à droite avec une quantité de 1.
  3. Ajuster la quantité : Utilisez les boutons + et - dans le panier, ou cliquez sur la quantité pour la saisir manuellement.
  4. Identifier le patient (optionnel) : Cliquez « + Patient » pour sélectionner ou créer un patient. Ses allergies seront vérifiées automatiquement.
  5. Choisir le mode de paiement : Espèces, Orange Money, MTN MoMo, Crédit (à terme), ou Assurance.
  6. Valider la vente : Cliquez « Valider ». Un reçu est généré. Le stock est automatiquement décrémenté.

Vente à l'unité (Déconditionnement)

Pour les produits configurés en déconditionnement, des boutons apparaissent :

  • Boîte — Vend la boîte entière
  • Plaquette — Vend une plaquette (sous-unité)
  • Unité — Vend un comprimé/gélule individuel

Vente à Crédit

  1. Sélectionnez le mode de paiement « Crédit »
  2. Identifiez obligatoirement le patient
  3. Choisissez la date d'échéance du paiement
  4. Validez. La dette apparaîtra dans l'Historique des Ventes.

Vente avec Assurance

  1. Sélectionnez « Assurance » comme mode de paiement
  2. Renseignez le nom de l'organisme (CNSS, ASCOMA, etc.)
  3. Entrez la référence de prise en charge
  4. Indiquez le montant couvert par l'assurance
  5. OrdiveX calcule automatiquement le ticket modérateur (la part patient)

Vente avec Ordonnance

  1. Activez le toggle « Ordonnance » en haut du panier
  2. Cliquez « Lier une ordonnance »
  3. Sélectionnez une ordonnance existante ou créez-en une nouvelle
  4. Les produits prescrits sont automatiquement ajoutés au panier
✅ Scan Code-Barres Utilisez la touche F9 ou cliquez l'icône 📷 pour activer le scanner. Sur mobile, la caméra arrière s'active automatiquement.
⚠️ Alertes automatiques au POS OrdiveX vous alertera automatiquement si :
  • Un produit est en rupture de stock
  • Un lot est périmé ou proche de l'expiration
  • Une interaction médicamenteuse est détectée entre deux produits du panier
  • Le patient a une allergie connue à un produit du panier

7. 💊 Catalogue Produits

Le catalogue contient tous les médicaments et produits parapharmaceutiques de votre pharmacie.

Ajouter un nouveau produit

  1. Cliquez « + Nouveau Produit »
  2. Renseignez : Nom commercial, DCI (molécule), Marque
  3. Sélectionnez la forme galénique : Comprimé, Sirop, Injectable, Gel, Tube, Sachet, etc.
  4. Entrez le Prix d'achat et le Prix de vente
  5. Configurez le stock minimum de sécurité (seuil d'alerte)
  6. Optionnel : Ajoutez la notice médicale, les précautions, les contre-indications
  7. Optionnel : Activez le déconditionnement (vente à la plaquette/unité)

Import massif CSV

Pour importer un catalogue complet (jusqu'à 50 000 produits) :

  1. Cliquez « Importer CSV »
  2. Sélectionnez votre fichier CSV (colonnes : name, dci, brand, form, buyPrice, sellPrice, minStock, category...)
  3. OrdiveX traite les produits par lots de 100 pour éviter tout blocage
  4. Un rapport d'import s'affiche à la fin

8. 📦 Gestion des Stocks

Vue d'ensemble

La page Stock affiche tous vos produits avec :

  • Quantité en stock (temps réel)
  • Stock minimum configuré
  • Statut : ✅ OK / ⚠️ Bas / 🔴 Rupture
  • Lots associés avec dates de péremption

Méthode FEFO

OrdiveX utilise automatiquement la méthode FEFO (First Expired, First Out) : lors d'une vente, c'est toujours le lot avec la date de péremption la plus proche qui est décrémenté en priorité.

Inventaire

  1. Recherchez le produit à inventorier via la barre de recherche (autocomplete)
  2. Cliquez sur le produit pour voir ses détails
  3. Cliquez « Inventaire » pour ajuster la quantité réelle
  4. Entrez la quantité comptée physiquement
  5. OrdiveX calcule et enregistre automatiquement l'écart (perte ou excédent)

Mouvements de stock

Chaque entrée ou sortie est enregistrée avec :

  • 📥 Entrée : Réception fournisseur, ajustement inventaire positif
  • 📤 Sortie : Vente, perte, destruction, ajustement négatif
  • Date, heure, utilisateur et motif de chaque mouvement

9. 📋 Ordonnances

Le module Ordonnances permet l'enregistrement numérique des prescriptions avec contrôle pharmaceutique.

Créer une ordonnance

  1. Cliquez « + Nouvelle Ordonnance »
  2. Patient : Sélectionnez un patient existant ou créez-en un. Si le patient a des allergies connues, elles sont affichées.
  3. Médecin : Entrez le nom, la spécialité, le N° d'Ordre et l'établissement du prescripteur.
  4. Date : Date de prescription et date de validité (3 mois par défaut).
  5. Médicaments : Cliquez « + Ajouter » pour ajouter chaque médicament prescrit avec quantité, posologie et unité.
  6. Options : Cochez « Renouvelable » si nécessaire. Ajoutez une note pharmacien si besoin.
  7. Photo : Prenez une photo de l'ordonnance papier originale (archivage 10 ans).
  8. Cliquez « Enregistrer en attente » ou « Valider & Enregistrer ».

Contrôle des interactions

OrdiveX vérifie automatiquement 30+ interactions médicamenteuses critiques entre les médicaments de l'ordonnance :

  • 🚨 Interactions graves (bandeau rouge) : Ex. Warfarine + Aspirine → Hémorragie
  • ⚠️ Interactions modérées (bandeau orange) : Ex. Fer + Ciprofloxacine → Absorption réduite
  • La note du pharmacien est pré-remplie automatiquement avec les interactions détectées

Dispenser une ordonnance

  1. Trouvez l'ordonnance dans la liste et cliquez 💊 Dispenser
  2. Si le patient a des allergies, une alerte de sécurité s'affiche
  3. Vous êtes redirigé vers le POS avec les produits de l'ordonnance déjà dans le panier
  4. Validez la vente normalement

10. 👤 Dossiers Patients

Créer un dossier patient

  1. Cliquez « + Nouveau Patient »
  2. Renseignez : Nom, Téléphone, Date de naissance, Sexe, Adresse
  3. Allergies : Très important ! Listez toutes les allergies connues (ex: Pénicilline, Aspirine). Le POS alertera si un produit allergène est ajouté au panier.
  4. Statut : Souscripteur principal ou Ayant Droit

Fiche patient

Chaque fiche patient contient :

  • Informations personnelles
  • Liste des allergies connues
  • Historique complet des achats
  • Ordonnances associées
  • Montant total des achats
💡 Recherche prédictive La liste des patients utilise l'autocomplete pour gérer efficacement les grandes bases (jusqu'à 20 000+ patients).

11. 🚚 Fournisseurs & Achats

Ajouter un fournisseur

  1. Cliquez « + Nouveau Fournisseur »
  2. Renseignez : Raison sociale, N° Agrément DNPM, Téléphone, Email, Adresse
  3. Configurez le délai de paiement (30 jours par défaut)

Passer une commande

  1. Sur la carte du fournisseur, cliquez « Commander »
  2. Sélectionnez les produits à commander avec les quantités
  3. Indiquez la date de livraison prévue
  4. Cliquez « Enregistrer » — La commande passe en statut « Brouillon »
  5. Cliquez « Envoyer » pour passer en statut « Envoyée »

Réceptionner une commande

  1. Allez dans Bons de Commande
  2. Trouvez la commande et cliquez « Réceptionner »
  3. Pour chaque article : confirmez la quantité reçue, le N° de lot et la date d'expiration
  4. Validez — Le stock est automatiquement mis à jour

Score fournisseur

Chaque fournisseur reçoit un score de fiabilité (sur 100) calculé automatiquement :

  • 40% — Taux de livraison à temps
  • 40% — Taux de complétion des commandes
  • 20% — Volume des commandes (bonus fidélité)

13. 📜 Historique des Ventes & Retours

Historique des Ventes

Toutes les ventes effectuées sont archivées avec :

  • Numéro de vente unique
  • Date et heure
  • Patient associé
  • Montant total et mode de paiement
  • Détail des articles vendus

Encaisser une dette

  1. Trouvez la vente à crédit (statut « Crédit ») dans l'historique
  2. Cliquez « Encaisser la dette »
  3. Entrez le montant reçu et le mode de règlement
  4. Le statut passe à « Payé »

Gérer un retour

  1. Trouvez la vente concernée et cliquez « Retour »
  2. Sélectionnez les articles à retourner (tout ou partie)
  3. Choisissez le motif : Erreur de dispensation, Péremption, Contre-indication, Autre
  4. Choisissez le mode de remboursement : Espèces, Mobile Money, Avoir
  5. Validez — Le stock est automatiquement réajusté à la hausse

16. 🛡️ Traçabilité & Pharmacovigilance

Le module Traçabilité permet de retracer l'intégralité du parcours d'un médicament :

  • Origine : Fournisseur, bon de commande d'origine
  • Lots : Numéros de lot, dates de fabrication et d'expiration
  • Mouvements : Chaque entrée et sortie du produit
  • Dispensation : À quels patients le médicament a été vendu
  • Ordonnances : Quelles prescriptions sont liées
💡 Conformité DNPM Ce module vous permet de répondre aux exigences de traçabilité de la Direction Nationale de la Pharmacie et du Médicament de Guinée.

20. ☁️ Synchronisation Cloud Multi-Appareils

Comment ça fonctionne

OrdiveX fonctionne en offline-first : toutes vos données sont stockées localement. Le cloud (Supabase) sert uniquement à synchroniser entre plusieurs appareils.

Configuration initiale

  1. Allez dans Paramètres → Appareil & Cloud
  2. Entrez l'URL Supabase et la Clé Supabase (fournis par TrillionX)
  3. Cliquez « Enregistrer la config Cloud »
  4. Donnez un nom à cet appareil (ex: « PC Bureau », « Mobile Vente »)

PUSH — Envoyer vers le Cloud

Les données se poussent automatiquement vers le cloud à chaque modification. Vous pouvez aussi forcer un envoi via Paramètres → « Envoi forcé vers le Cloud (PUSH) ».

PULL — Récupérer du Cloud

Pour récupérer les données d'un autre appareil, cliquez « Récupérer les données du Cloud (PULL) » dans les Paramètres.

⚠️ Important Si vous travaillez hors-ligne depuis longtemps, pensez à faire un PULL dès que vous retrouvez la connexion pour récupérer les modifications des autres appareils.

Moniteur de Réseau

Cliquez l'icône 📡 dans la barre supérieure pour voir tous les appareils connectés à votre pharmacie, leur statut (en ligne/hors ligne) et la dernière synchronisation.

23. 🛠️ Dépannage & FAQ

L'application est lente

  • Fermez les onglets/apps inutiles
  • Rafraîchissez avec Ctrl+Shift+R
  • Si vous avez 50 000+ produits, la recherche utilise l'autocomplete pour rester fluide

Les données ne se synchronisent pas

  1. Vérifiez votre connexion internet
  2. Allez dans Paramètres → Synchronisation
  3. Cliquez « Réparer l'envoi Cloud » — Cela force un renvoi complet
  4. Puis faites un PULL pour récupérer les données manquantes

J'ai perdu des données

  1. OrdiveX fait des sauvegardes automatiques toutes les 30 minutes
  2. Allez dans Paramètres → Restaurer une sauvegarde
  3. Sélectionnez le fichier JSON de sauvegarde
  4. OrdiveX téléchargera d'abord un fichier de secours de vos données actuelles avant de restaurer

Le POS ne décompte pas le stock

  • Vérifiez que le produit a bien des lots avec du stock (Gestion des Stocks → cliquez sur le produit)
  • Si le stock est à 0, le produit est en rupture et OrdiveX vous l'indique

L'imprimante ne fonctionne pas

  • Vérifiez le Bluetooth de votre appareil
  • Assurez-vous que l'imprimante est allumée et appairée
  • Essayez d'imprimer un test depuis les paramètres Bluetooth de votre téléphone
